With you can clear your 3Ds Max scene from accumulated garbage.

Features:

  • Clear Garbage Collection
  • Clear Memory
  • Clear Reactor Collision
  • Clear Animated Layers
  • Clear Missing Bitmaps
  • Remove Empty Layers
  • Remove All Animated Keys
  • Remove Missing Objects
  • Remove Invalid Objects (ParticleView)

Более подробно о каждом типе очистки читайте ниже.

Track Note

Удаляет так называемые Note Tracks, - это служебная информация об анимационых ключах. Note Tracks появляются при Merge, Import, X-Ref моделей или же автоматически и может значительно увеличивать размер сцены.

Garbage

Удаляет Local Garbage Collection, - который со временем накапливается в сцене.
При уменьшении виртуальной памяти, он очищается автоматически, но до этого времени мусор будет накапливаться в сцене и увеличить ее размер.
Эта опция поможет очистить Garbage collection немедленно!

Memory

Очищает Undo Buffer, - историю последних действий.
А также удаляет Bitmap Cache, - кэш растровых изображений.

Примечание!

После использования этой опции, у вас могут перестать отображаться текстуры во вьюпорте!

Reactor Collision

Эта опция актуальна для старых версий 3D Studio Max.

Anim Layers

Удаляет анимационные слои, которые увеличивают размер сцены. Данные слои могут создаваться автоматически, даже если у вас статическая сцена, без анимации.

Missing Objects

Удаляет объекты созданные внешними плагинами, которые не подключены и объекты, которые не корректно были импортированы в сцену.
Эта опция также удаляет пустые объекты.

Missing Bitmaps

Удаляет пути для несуществующих текстур. Эта опция помогает избавиться от ошибки Missing External Files при открытии сцены.

Предупреждение!

Будьте внимательны, если вы перенесли сцену или создали новую и не успели присвоить пути текстур через Utilites → Bitmap/Phonometric Paths, вы можете сбросить все пути текстур!

Empty Layers

Удаляет слои, в которых нет объектов.
Иногда при импорте чертежей из AutoCAD или Merge моделей из 3ddd.ru/3dsky.org в сцене могут создаваться нежелательные пустые слои. Эта опция поможет удалить их.

Anim Keys

Удаляет все анимационные ключи.

Предупреждение!

Если у вас в сцене содержится нужная анимация, отключите эту опцию! Иначе вся анимация будет удалена!

Particle View (Invalid Objects)

Удаляет пустые объекты с названием Particle_View_001...
Если у вас в сцене нет Particle анимации, вы можете смело использовать эту опцию.
Run the script from ToolBar or QuadMenu (depending on the settings during installation script).

Select the required items for each type of scene cleaning that you consider necessary.

Warning!

It is strongly recommended before using save and make a backup of the scene!
After cleaning, the action script can not be undone!
We recommend that you carefully review the information in this section!
Then press the Start.

Note!

All settings of the selected types of cleaning are saved after closing 3D Studio Max.

After a while, after the end of the process, on the other hand each type of cleaning you will see a colored status: 
CLEARED - found and deleted garbage for this type.

NOT FOUND - not found garbage for this type.

DISABLED - this option has been disabled.
For more information about each type of treatment, see below.

Track Note

Note Tracks appear at Merge, Import, X-Ref models or automatically, and can significantly increase the size of the scenes.
This option will remove them.

Garbage

When memory run low garbage cleared automatically, but until that time, the garbage will accumulate in the scene and to increase its size.
This option will help clean Garbage collection immediately!

Memory

It clears the Undo Buffer, - history of recent actions.
Also removes the Bitmap Cache.

Note!

After using this option, you may no longer be displayed texture in the viewport!

Reactor Collision

This option is relevant for older versions of 3D Studio Max.

Anim Layers

Removes animation layers that increase the size of the scene. These layers can be created automatically, even if you have a static scene, without animation.

Missing Objects

Removes objects by external plugins, which are not connected, and objects that are not correctly imported into the scene.
This option also removes blank objects.

Missing Bitmaps

Removes the path for non-existent textures. This feature helps to avoid errors Missing External Files at the opening scene..

Warning!

Be careful if you transferred or created a new scene and did not have time to assign textures the path through Utilites → Bitmap / Phonometric Paths, you can reset all the paths textures!

Empty Layers

Removes layers in which no object.
Sometimes when you import drawings from AutoCAD or Merge 3ddd.ru/3dsky.org models of the scene can be created unwanted empty layers. This option will remove them.

Anim Keys

Removes all animation keys.

Warning!

If you have in the scene contains the desired animation, uncheck this option! Otherwise, all the animation will be removed!

Particle View (Invalid Objects)

Removes empty objects named Particle_View_001 ...
If you have in a scene no Particle animation, you can safely use this option.
